              flowblok Mod Yves Baumes • 2 years ago

              The simplest example is a shell script using bash.

              I actually have no idea what happens for subshells, perhaps someone wants to test it out and let me know?

              As for a definition of an interactive shell, here’s what the bash man page says:
              An interactive shell is one started without non-option arguments and without the -c option whose standard input and error are both connected to terminals (as determined by isatty(3)), or one started with the -i option.
              (and of course, all other shells are therefore non-interactive)
